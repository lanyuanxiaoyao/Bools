# Btools
utools 插件, 用于提供一些与浏览器相关的功能, 为隐藏或需要多步操作的功能提供快捷方式.

*特别说明: 由于不同的浏览器支持程度不同, 一个功能不一定能在所有浏览器中都能实现.*

# 版本

  1. 实现转换当前网址为二维码
  2. 快捷打开下载, 插件, 商店等页面
